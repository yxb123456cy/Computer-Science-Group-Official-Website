## 项目介绍

本项目是为高校计算机科技小组打造的一站式静态官网，旨在展示社团风采、沉淀技术成果、吸引新生加入、提供实用资源导航。项目基于 Vue3 + TypeScript + Vite 构建，采用 ArcoDesign Vue 组件库实现科技感视觉界面，已支持 Vercel & Render 双平台一键部署，开箱即用。

## 核心功能

1.  社团全景展示：包含简介、组织架构、发展历程、荣誉资质等基础信息
2.  活动动态呈现：常规活动回顾、特色活动展示、近期活动预告
3.  技术成果沉淀：社团自研项目、竞赛获奖项目、成员优秀作品展示
4.  新生招新入口：明确招新要求、报名渠道、常见问题解答
5.  实用资源导航：编程学习、开发工具、竞赛资料等资源汇总
6.  多渠道沟通：官方账号二维码、反馈表单、联系方式展示

## 技术栈

| 分类      | 技术选型                       |
| --------- | ------------------------------ |
| 前端框架  | Vue 3（Composition API）       |
| 类型校验  | TypeScript                     |
| 构建工具  | Vite                           |
| UI 组件库 | ArcoDesign Vue                 |
| 样式方案  | CSS3 / SCSS（支持主题定制）    |
| 部署平台  | Vercel（优先）、Render（备用） |
